About Me

This is the tale of a 26-year-old girl with a dream that sparked at 14. My heart longed to be a software developer, but my parents envisioned a tennis star. So, I ventured into the world of tennis, but my passion never waned.

Then, everything changed with Covid-19. I saw a chance to blend my tennis skills with technology, fulfilling my lifelong dream of becoming a software developer. Today, I'm a junior at Tiffin University, pursuing Computer Science with a double concentration in Software Development and Web Design, holding a remarkable 4.00 GPA.

My passion for programming languages drives me to continuously expand my knowledge and skills. I am on a path of continuous growth and am excited about the endless possibilities that the world of software development has to offer.

My Projects

Birthday Reminder App

A Birthday Reminder application built with Next.js, utilizing the power of server-side rendering and React. MongoDB is used as the NoSQL database for storing user and birthday data, and Mongoose is employed as the Object Data Modeling (ODM) library for MongoDB. The app also incorporates NextAuth for authentication, SendGrid for email notifications, and is deployed and hosted on Vercel.

React Meetups App

The React Meetups App showcases local meetups using React and Next.js. CSS Modules are employed for styling, providing scoped styling to the components. The app is deployed and hosted on Vercel, offering seamless accessibility to users.

Todo App with Typescript

The Todo App with Typescript is a task management application developed with TypeScript and React. It provides an organized and efficient way to manage tasks, allowing users to add, update, and remove tasks effortlessly. CSS Modules are used for styling, ensuring component-specific styles. The project is version-controlled using Git and hosted on GitHub.
